Opdracht 8: Wat is een strict equality comparison ?

Een strict equality comparison is een vergelijking met drie = tekens, waarbij niet alleen de waarde moet kloppen maar ook de vorm waarin het gepresenteerd wordt, zo is de numericale inhoud van een String niet gelijk aan de Number value die hetzelfde is.
