Voor deze opdracht is het handig om in te lezen over methods van String. Aan de linkerkant staat een menubalk. Onder de categorie methods staan een lijst met functies die je met een String kan aanroepen. Maak een eigen voorbeeld van de volgende functies en licht toe wat de functie doet in het blok:
1. charAt()
charAt laat je een specifieke karakter selecteren binnen de string, in ons geval laat 3 de vierde character selecteren(0,1,2,3).
2. concat()
Met concat kan je meerdere strings aaneenrijgen. 3. slice()
Met slice kan je een segment selecteren door de eerste en laatste character te benoemen.
4. split()
met split kan je een deel van String selecteren door aan te wijzen in wat voor soort "lijst/indeling" gezocht moet worden.
5. substr()
Met substr kan je ook een deel selecteren, je geeft het begincharacter aan en eventueel hoeveel vervolgcharacters, anders selecteert hij tot het einde.
