HTML Elementen

Bij het inladen van een bestand is het noodzakelijk om de locatie van het bestand aan te geven. Dit kan op 2 manieren: ● Absolute path ● Relative path Wat houden deze 2 paths in?

Een absolute path is het volledige adres naar een bepaald bestand toe(meestal links) en de relative path is de path relatief ten opzichte van de folder waarin het HTML bestand staat.

Een image element is voor het inladen van een afbeelding. Afbeeldingen kunnen van verschillende formaat zijn. Bijvoorbeeld, een afbeelding kan een JPG formaat zijn. 2. Zoek uit welke formaat afbeeldingen er nog meer kan zijn.

5. Wat houdt de alt attribute in? En waarom is het handig om de alt attribuut te gebruiken?

De alt attribute geeft een naam of een stukje tekst die de afbeelding omschrijft. Dat is handig in het geval dat de afbeelding om wat voor reden dan ook niet weergegeven wordt, zo heeft de user alsnog een idee wat het zou moeten voorstellen.

7. Speel een audio bestand af met source tag. Leg uit wat hier het voordeel van is.

Je kan verschillende sources in verschillende formats toevoegen waardoor een browser die x format niet herkent naar de alternatieve y format kan gaan.

Niet alle formaten worden ondersteund door de browser. Hiervoor is een oplossing voor. Speel een video bestand af met source tag. Leg uit wat hier het voordeel van is.
