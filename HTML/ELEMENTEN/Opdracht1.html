1. Wat is het verschil tussen het internet en het world wide web?

Het internet is het gigantische, fysieke netwerk van computers, servers en alle toebehoren, zowel bedraad als draadloos en het www is een van de (meest voorkomende)manieren om toegang te krijgen tot het internet, en het gebruik van HTML en HTTP onderscheidt het van de andere manieren.

2. Wat is het verschil tussen HTML, CSS en JavaScript?

HTML is de fundamentele taal die je gebruikt om iets op het www te weergeven, CSS is hoe je die taal wilt vormgeven en stylen en Javascript gebruik je om de eerste twee te gebruiken en te manipuleren.

3. Wat is een HTML tag en een HTML element? Hoe open je een HTML tag en hoe sluit je een HTML tag. Maak hiervoor voorbeelden en leg uit.

Een HTML tag is het kopje voor en/of na je daadwerkelijke content waardoor programma's kunnen herkennen wat het is en waar ze bepaalde styling op kunnen toepassen. Het HTML element is een complete set van twee tags, 1 om te openen en een om te sluiten met daartussen de content die je wilt laten zien. De begintag vangt aan met < waarna de identificerende HTML code komt(bv p of h3) gevolgd door een >. De eindtag is hetzelfde maar er komt een / na de <= < p > hallo < /p > (het voorbeeld heeft extra spaties om niet als daadwerkelijke tag gelezen te worden en meer voorbeelden zijn te zien in de HTML file van deze site :) )
