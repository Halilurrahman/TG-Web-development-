HTML tables

10. Om een tabel te maken heb je volgende elementen nodig: ● < table> ● < thead> ● < tbody> ● < tfoot> ● < th> ● < tr> ● < td> Geef voor elke element een definitie.

Het table element geeft aan dat het om een tabel gaat, thead groepeert alle koppen bij elkaar, tbody geeft aan waar de bulk van de informatie zit en de tfoot geeft de onderkant van de tabel weer(waar meestal de cumultatieve resultaten staan). Th geeft de headerinfo aan binnen 1 cel, tr geeft een rij binnen de tabel aan, en td geeft een individuele datacel aan.
