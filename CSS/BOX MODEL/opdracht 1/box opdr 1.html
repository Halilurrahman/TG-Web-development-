Wat zijn de eigenschappen van block boxes en inline boxes?

Block boxes zijn elementen die;

  1. op een nieuwe regel beginnen
  2. block boxen vullen de beschikbare ruimte in de breedte op
  3. width en height properties worden gerespecteerd
  4. Padding, margin and border duwen andere elementen weg als ze worden gewijzigd

Inline boxes zijn elementen die;

  1. niet op een nieuwe regel beginnen
  2. width en height properties zijn niet van toepassing
  3. Verticale padding, margins, en borders kunnen worden toegepast maar beïnvloeden andere inline boxes niet
  4. Horizontale padding, margins, en borders kunnen worden toegepast en beïnvloeden andere inline boxes wel

If a box has an outer display type of inline, then: The box will not break onto a new line. The width and height properties will not apply. Vertical padding, margins, and borders will apply but will not cause other inline boxes to move away from the box. Horizontal padding, margins, and borders will apply and will cause other inline boxes to move away from the box.
