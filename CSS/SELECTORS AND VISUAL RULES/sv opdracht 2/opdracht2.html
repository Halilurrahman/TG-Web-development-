Opdracht 2

CSS pas je toe met een bepaalde syntax. Hoe ziet zo'n syntax eruit? Kun je ook uitleggen wat elk element uit de syntax betekent?

De CSS syntax is op te delen in selectors en declaration. de declarations zijn op te delen in properties en values. De selector geeft aan op welke HTML element de selector van toepassing is, de declaration(s) geven aan hoeveel stylingselementen er zijn, de property geeft aan welk stylingselementn(bijvoorbeeld kleur, lettertype, grootte etc ) er gebruikt moet worden en de value geeft aan wat de waarde van dat element is.
