Maak van de bovengenoemde selectors voorbeelden. Geef in je antwoord ook aan wat de voordelen zijn van elke selector.

De type selector selecteert (alle) complete HTML elementen en is in die zin goed om (basale) styling gelijk te trekken over een geheel document. Descendant selectors zijn selectoren binnen selectoren, bijvoorbeeld ul li{} waarbij ieder li element die afstamt van een ul element de styling krijgt. Descendant selectors zijn handig om specifieker(elementen binnen) elementen documentbreed te stylen. Class selectors zijn selectors die handmatig zijn aangemaakt en die je toepast op welke element of combinatie van elementen die je maar wilt.
