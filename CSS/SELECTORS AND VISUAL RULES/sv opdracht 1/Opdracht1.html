Opdracht 1

Je kunt op 3 verschillende manieren CSS in je HTML inladen. Op welke 3 manieren kun je dat doen? Leg ook voor elke wijze uit wat het precies inhoud.

Er zijn 3 soorten CSS, Inline CSS, Internal CSS en External CSS. Inline CSS is styling binnen een HTML element, en is alleen van toepassing op het element waarbinnen het zit. Internal CSS is styling bovenaan een HTML bestand in de head tag, het is toepasbaar op alle elementen waar het naar verwijst, zonder dat je individuele Inline CSS nodig hebt. External CSS is vergelijkbaar met Internal CSS alleen staat alles in een eigen bestand in tegenstelling tot de headtag, het voordeel hiervan is de overzichtelijkheid van je HTML bestand en dat het makkelijker wordt om de twee bestanden naast elkaar in te zien.
